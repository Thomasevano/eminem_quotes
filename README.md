# Eminem Quotes

A little web app to show a random Eminem quote from a JSON file (api/quotes.json).

## Getting started

### Clone the Repo

```
git clone https://github.com/Thomasevano/eminem_quotes.github.io.git
```

## Install

```
npm install
```

## Start

```
npm start
```

## Build

```
npm run build
```
